# ITransition Intern
## _Task 6_

Simple Chat application using:

- React
- NodeJs
- MongoDB

### Web-Page

https://chattask5.herokuapp.com/

## Installation

Install the dependencies and devDependencies and start the server.

To launch client side
```sh
cd client
npm i
npm start
```

To launch server side
```sh
cd server
npm i
npm start
```
